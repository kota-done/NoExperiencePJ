# 仮機能メモ

## 目的
- 従業員管理サイトとして最低限のサイトらしさを画面上に追加する。
- バックエンド未実装の機能を、今後のカリキュラム課題として扱えるよう整理する。

## 今回 UI 上に追加する仮機能
- ログインボタン
  - 画面上は表示するが、認証処理や遷移先は未実装
- 戻るボタン
  - 登録画面、更新画面から一覧画面へ戻る導線として表示
- ホーム導線
  - 一覧画面へ戻るための共通ナビゲーション
- ダッシュボード風の補助アクション
  - 検索、通知、レポートなどを将来拡張できる想定でダミー表示

## 未実装のまま残す内容
- 認証/認可
- ログイン後セッション管理
- 検索条件入力と検索処理
- 通知機能
- レポート出力
- 権限別メニュー制御

## 課題化しやすいテーマ
- ログイン画面追加と認証処理実装
- 共通ヘッダーの部品化
- 一覧画面への検索フォーム追加
- 通知一覧画面の新規作成
- レポート画面または CSV 出力機能の追加
- エラーメッセージ表示や入力バリデーションの実装

## 補足
- 本メモは、現時点で画面に先行配置した要素と未実装領域を区別するためのもの。
- 実装時は、本メモの各項目を個別課題として切り出せる。
