# F006 派遣先企業一覧機能

## 1. 概要
- 登録済み派遣先企業を一覧で表示する。

## 2. 主フロー
1. 利用者が派遣先企業一覧画面にアクセスする。
2. Controller が Service に一覧取得を依頼する。
3. Service が Repository 経由で派遣先企業一覧を取得する。
4. Controller が Model に一覧情報を設定する。
5. 一覧画面テンプレートを返却する。

## 3. 入出力
- 入力: なし
- 出力:
  - 派遣先企業一覧
  - 企業名
  - 連絡先情報

## 4. 責務分担
- Controller: 画面表示データの組み立て
- Service: 一覧取得処理
- Repository: `client_companies` からの一覧取得
