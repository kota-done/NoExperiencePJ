# F009 派遣先企業削除機能

## 1. 概要
- 指定した派遣先企業を削除する。
- ただし、従業員が紐づいている場合は削除不可とする。

## 2. 主フロー
1. 利用者が削除操作を実行する。
2. Controller が ID を Service に渡す。
3. Service が所属従業員の有無を確認する。
4. 所属従業員がいない場合、Repository 経由で削除する。
5. 一覧画面へリダイレクトする。

## 3. 例外フロー
- 所属従業員が存在する場合、削除せずエラー扱いとする。

## 4. 入出力
- 入力: `id`
- 出力: 一覧画面へ遷移

## 5. 責務分担
- Controller: 受け付けと遷移制御
- Service: 削除可否判定と削除処理実行
- Repository: `client_companies` から削除
