# F003 従業員更新

## 1. 概要
- 既存従業員情報を更新する。

## 2. 主フロー
1. 利用者が更新画面を開く。
2. Controller が ID 指定で Service から従業員情報を取得する。
3. 更新画面に既存値を表示する。
4. 利用者が修正して送信する。
5. Service が Repository 経由で更新保存する。
6. 一覧画面へリダイレクトする。

## 3. 例外フロー
- 対象 ID が存在しない場合、Service が例外を送出する。

## 4. 責務分担
- Controller: 既存データ表示と遷移制御
- Service: 対象取得と更新保存
- Repository: `employees` の更新
